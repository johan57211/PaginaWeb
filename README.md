# Foro Hollow Knight

## URL del Proyecto
https://paginaweb-glpr.onrender.com/

## Descripción
Aplicación web multipágina dedicada al videojuego Hollow Knight, diseñada como un foro interactivo para la comunidad de fanáticos. El sitio ofrece una experiencia inmersiva con diseño responsive en blanco y negro que refleja la estética del juego. Incluye secciones de historia y lore, guías detalladas de 12 jefes con modales interactivos, galería de fanart, noticias actualizadas sobre Silksong y el juego original, y un formulario de contacto con validación en tiempo real. Implementa un carrusel personalizado de personajes, navegación sticky adaptativa y efectos visuales avanzados con animaciones CSS.

## Lenguajes y Tecnologías Utilizadas
- **HTML**: Estructura semántica del contenido de las 6 páginas web, incluyendo formularios, tablas y elementos multimedia.
- **CSS**: Estilos avanzados con diseño responsive, transiciones, gradientes y efectos visuales que reflejan la estética del juego.
- **JavaScript**: Interactividad del lado del cliente, implementando carrusel de personajes con autoplay, modales dinámicos para guías de jefes y validación en tiempo real del formulario de contacto.
- **Python**: Lenguaje de programación del backend, manejando la lógica del servidor y configuración de la aplicación.
- **Flask**: Framework web de Python para el manejo de rutas, renderizado de templates HTML y despliegue de la aplicación.
